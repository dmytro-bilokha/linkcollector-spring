Using Linkcollector

  1. On the homepage enter the search query.
  2. Fill the tags table.
  3. Press Search button.
  4. Linkcollector will send your search query to the Bing search engine. Eache result from the Bing will be scanned for presents of the tag's text. If tag's text is present on the result page, the page score value will be increased on the tag's weight.
  5. All processed Bing results will be shown sorted by the score value.
  6. You can click Edit search link to edit search parameters and repeat it. Or you can just click HOME to accomplish a new search.
